The MLA RDA Application Profile

An application profile conforming to the recommendations in the MLA Best Practices is available at TEMPORARY LINK. The profile is an Excel spreadsheet divided into multiple worksheets: one for elements common to all musical resources and additional sheets for elements (or treatments of elements) specific to scores and recordings (in physical or electronic formats).
